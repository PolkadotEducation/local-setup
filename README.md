# Quickstart local development setup

### 1 - Clone this repo
```
git clone https://github.com/PolkadotEducation/local-setup.git
cd local-setup
```

### 2 - Clone all other repos
```
./run.sh
```

### 3 - Run Docker compose
```
docker compose up
```

### 4 - Shutdown Docker compose
```
docker compose down
```
